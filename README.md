# Termino 💻💬

A real-time terminal chat application built with Node.js and Socket.IO.

## 🚀 Features
- Real-time messaging
- Username mentions (@user)
- Desktop notifications
- Join / leave alerts
- Group chats

## 📦 Installation

```bash
git clone https://github.com/nikamapurva25-webgit
cd termino
npm install
