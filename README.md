# Termino 💻💬

A real-time terminal chat app built with Node.js and Socket.IO.

## 🚀 Features
- Real-time messaging
- Username mentions (@user)
- Desktop notifications
- Join / leave alerts
- Group chats

## 📦 Installation

git clone https://github.com/YOUR_USERNAME/termino.git  
cd termino  
npm install  

## ▶️ Run

Start server:
npm start  

Run client:
npm run client
